  • 1：A new page pops up, click the get button, and allow the download of configuration files

  • 2： After the configuration file is downloaded, click Close and allow

  • 3：Automatically enter the description file and device management, select union century

  • 4：Enter the installation description file, click the install button, and select the install button below.

  • 6：The installation pop-up window appears, click Install. Start to install the app

  • 7：In the display downloading, you can directly go back to the desktop to view the download progress and wait for the download to complete,then enjoy
